## 2. 수정하고싶은 사항
- 재료 탭에서 노출되는 재료들의 칸 크기가 너무 작아서 불편함 -> 인게임 오브젝트로 변경하면서 위치와 크기 재조절
- 임시레시피 띄워뒀지만 작아서 보기 불편함 -> 추후 도감에서 크게 노출
- 현재 재료를 눌러서 조리대에 넣은게 무엇인지 확인 안되서 테스트하기에 불편함 -> 추후 재료 누를때마다 텍스트 노출 또는 조리대 안에 누른 재료 표시 

## 3. 발견된 버그
- 요리 실패 시 간헐적으로 "실패" 이미지가 안나오는 현상
- 플레이 중 간헐적으로 음식이 손님위에 떠져있고 일정시간 있다가 사라지는 현상

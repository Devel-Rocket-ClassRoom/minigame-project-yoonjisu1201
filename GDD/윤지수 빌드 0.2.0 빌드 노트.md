## 1. 구현된 기능 목록

- 영업씬에서 세션 골드 UI 노출, 영업 마감 UI에 골드 획득량 표시
- 로비 씬에서 총 골드 UI 노출, 씬 전환 시 토탈 골드 유지

### ✅ 메타 진행 — GameContext 도입 (#34)

- 골드 계산, 손님 스폰 등 전역 값 하드코딩 제거

### ✅ 메타 진행 — 푸드트럭 등급 시스템 (#35)

### ✅ 메타 진행 — 해금 시스템 (#41)

- 방명록: 손님 최초 드롭 시 해금
- 손님이 주문할 때 해금된 메뉴만 주문하도록 수정
- 모든 손님·레시피 SO 에셋 생성 완료

- 요리 / 손님 / 유물 탭 구성
- 해금 항목: 이미지 + 설명 표시 / 미해금 항목: 실루엣 표시

### ✅ 업그레이드 시스템 (#49, 부분 구현)
- 업그레이드 항목 확정 (5개 섹션):
  - 조리대 슬롯 수 증가
  - 조리 속도 단축
  - 조리대에 넣은 재료 노출 기능
  - 주문 말풍선 힌트 기능
  - 영업 시작 시 레시피 재료 미리 담기 슬롯
- `UpgradeManager`: 업그레이드 비용 관리, 섹션별 현재 레벨 관리, 업그레이드 가능 여부 메서드, 골드 차감
- `UpgradeMenuUI`: 섹션별 텍스트 정보 불러오기, 현재 골드 / 최대 레벨 체크, 레벨업 이벤트 등록
- 임시 업그레이드 UI 패널 연결: 현재레벨/최대레벨 표시, 골드 소모, 최대 레벨 시 버튼 잠금
- 조리대 슬롯 수 조절, 조리 시간 감소 동작 확인

### ✅ 조리 시스템 개선
- 조리대에 재료 추가 시 1초 노출 후 사라지는 연출 추가 (위로 살짝 올라가며 흐릿하게 소멸, 코루틴 사용)
- 조리대 최대 재료 수 4개로 제한
- 조리 시작 버튼 UI, 버리기 UI → 인게임 오브젝트로 전환

### ✅ 버그 수정

## 3. 플레이에 필요한 정보

### 기본 조작
1. 로비 씬에서 **영업 시작** 버튼 → 게임 씬 전환
2. 조리대 양옆 재료 오브젝트 탭 → 활성 슬롯에 재료 순서대로 투입 (최대 4개)
3. **조리 시작 오브젝트 탭** → 타이머 시작
4. 타이머 종료 → 슬롯에서 완성 음식 드래그 → 손님에게 전달
5. 서빙 성공 → 골드 적립, 경험치 누적
6. 세션 종료 → 마감 UI (획득 골드·경험치 확인) → 로비 복귀
7. 로비에서 **도감** 아이콘 → 해금된 재료·레시피·손님·유물 확인
8. 골드 보유 시 **업그레이드** 패널에서 조리대 슬롯 수 / 조리 속도 업그레이드 가능

### 주의사항
- 레시피 순서가 중요합니다. 재료 종류가 맞아도 순서가 틀리면 망한 음식이 됩니다.
- 전용 메뉴는 단골 본인이 주문했는지에 따라 마지막 재료가 달라집니다.
- 트럭 등급에 따라 방문 손님과 해금 레시피가 달라집니다.
- 업그레이드 힌트 기능, 재료 슬롯 기능은 아직 미구현입니다.

### 현재 플레이 가능한 범위
> 로비 → 영업 시작 → 손님 등장/주문 → 재료 투입 및 조리 → 완성 음식 드래그 서빙 → 골드·경험치 적립 → 세션 마감 → 트럭 등급 상승 및 해금 확인 → 로비 도감·업그레이드 → 반복

# 별빛 푸드트럭 (Starlight Food Truck)

*버전: 1.7 (해금 구조 확정 - 11등급 시스템 + 유물 5개 누적)*

- 장르: 감성 힐링 요리 타이쿤 (Cozy Tycoon)
- 플랫폼: 모바일 (iOS / Android) · 가로 화면 · 한 손/두 손 조작 호환
- 핵심 목표: 평일 15일 내 완성 가능한 '기능 중심'의 1인 개발 프로젝트 완수

---

## 1. 게임 개요

**로그라인**: 별빛이 내리는 숲속에서 유령 손님들에게 완벽한 한 끼를 대접하고, 그들의 방명록과 유물을 수집하며 나만의 푸드트럭을 성장시키는 푸드트럭 운영기.

**핵심 가치**:
- **조작의 재미**: List 기반 순차 검증 조리 시스템과 명확한 피드백
- **수집의 재미**: 유령들의 방명록(서사)과 영구 패시브 유물(기능)을 모아가는 성취감
- **성장의 재미**: 트럭 등급 상승을 통한 콘텐츠 해금과 골드 업그레이드의 조화

---

## 2. 플랫폼 및 조작

- 플랫폼: 모바일 (iOS / Android)
- 엔진: Unity 6.3
- 화면 방향: 가로 (Landscape)
- 주 조작: 탭(Tap) 중심 + 완성 음식 전달은 드래그
- 조작 자세: 한 손/두 손 모두 호환 — 핵심 UI는 하단·양 가장자리 배치
- 보조 입력: 롱프레스(유령 정보/도감 상세) — 차후 확장 여지로 유보

---

## 3. 서사 및 세계관

### 3.1 시놉시스

안개 낀 숲길에서 길을 잃은 주인공은 돌아가신 할머니가 남긴 낡은 푸드트럭과 미완성 레시피 북을 발견합니다. 할머니는 생전에 숲을 지나는 영혼들에게 온기를 나눠주던 요리사였습니다. 주인공은 영혼들에게 음식을 대접하며 할머니의 진심을 이해하고, 백지로 변해버린 도감을 완성하여 현실로 돌아갈 용기를 얻습니다.

### 3.2 주요 등장인물

- **플레이어**: 할머니의 유산을 이어받은 임시 운영자
- **영혼 손님**: 총 10종 — 기본 길 잃은 영혼 1종 + 단골 영혼 9종(최종 단골 = 기다리는 할머니 포함)

기본 길 잃은 영혼은 가장 빈도 높게 등장하는 표준 손님이며, 단골들은 각자의 사연·시그니처 소품·고유 유물을 가집니다.

## 4. 핵심 게임 시스템

### 4.1 핵심 루프

```
[영업]
  → [손님 만족 → 트럭 EXP 누적]
  → [등급 상승 시 신규 유령·레시피 해금]
  → [유물 드롭]
  → [유물 1개 = 방명록 해금 / 5개 = 기능적 보상]
  → [골드로 업그레이드]
  → [도감 완성 및 엔딩]
```

### 4.2 영업 세션 구조

- **세션 길이**: 2~3분 (모바일 캐주얼 미니게임 권장 길이)
- **세션 종료 조건**: 전체 세션 타이머 만료 시 자동 마감 → 정비 시간(유물 적용 등) 진행
- **손님 인내심**: 각 손님은 자신만의 인내심 게이지를 가지며, 게이지 소진 시 화내며 떠남
- **페일 정책 (Soft Fail)**: 손님 인내심 소진 또는 조리 오답 시 — 해당 손님의 골드 수익만 받지 못하고 다음 손님으로 진행. 하드 페일 없음

### 4.3 조리 시스템

#### 4.3.1 자료구조 결정

- 각 슬롯은 재료 **List**를 보유. 재료 push 순서가 그대로 레시피 검증 순서가 됨.
- Stack(LIFO)이 아닌 List(시간순)로 결정 — '재료 투입 순서의 정확성'이 핵심이므로 가독성과 직관성을 우선.
- 레시피도 ScriptableObject 내부에 재료 List로 정의 → 슬롯의 List와 일괄 비교.

#### 4.3.2 조리 프로세스 (탭 흐름)

1. 빈 슬롯 탭 → 활성 슬롯으로 전환 (테두리 글로우 등 시각 강조)
2. 재료 패널의 재료 버튼 탭 → 활성 슬롯의 List에 순차적으로 push, 슬롯 위에서 통통 튀는 연출
3. 다른 슬롯도 사용하려면 해당 슬롯 탭으로 활성 전환 (멀티태스킹)
4. '조리 시작' 버튼 탭 → 해당 슬롯의 타이머 가동 (익히는 시간)
5. 타이머 종료 → 슬롯이 '수령 가능' 상태로 변경, 시각적 알림(반짝임)
6. 수령 시점에 레시피 일괄 검증 → 일치 시 완성 음식, 불일치 시 망한 음식 팝업
7. 완성 음식을 손님에게 드래그하여 전달 → 골드 획득 + 트럭 EXP +1

#### 4.3.3 슬롯 운영 규칙

- 기본 슬롯 2개, 푸드트럭 업그레이드로 +2 추가 가능 (최대 4개)
- 각 슬롯은 독립적으로 동작하며, 활성 슬롯은 한 번에 하나만 지정 가능
- 조리 중도 취소 가능 (재료는 무한 재고이므로 손실 없음)

#### 4.3.4 자원 관리

- **재료**: 무한 재고 — 재료 버튼 제한 없이 탭 가능 (관리 부담 최소화, 조리 자체에 집중)
- **재료 해금**: 신규 레시피 해금 시 해당 레시피에 필요한 재료가 자동으로 패널에 추가됨
- **재료 원가 없음**: 모든 수익은 '완성 음식 판매 가격'에서 발생

#### 4.3.5 예외 처리 (감성 힐링 톤 유지)

| 상황 | 처리 방식 | 디자인 의도 |
|---|---|---|
| 레시피 불일치 (재료/순서 오답) | 수령 시점에 '망한 음식' 팝업, 골드 0 | Soft Fail — 영업은 계속됨 |
| 조리 도중 취소 | 가능, 재료 무한이므로 무손실 | 실험적 플레이 허용 |
| 잘못된 손님에 드래그 | 음식이 슬롯으로 자동 복귀 (무손실) | 조작 실수에 관대 |
| 완성 후 방치 | 일정 시간 후 음식 상함 처리 (망한 음식 전환) | 긴장감 + 슬롯 회수 유도 |
| 완성 전 손님 이탈 (인내심 0) | 조리는 그대로 진행, 같은 메뉴 다음 손님에게 재활용 가능 | 음식 버리지 않는 따뜻한 디자인 |

### 4.4 해금 및 진행 구조

#### 4.4.1 트럭 등급 시스템 (Truck Rank)

- 총 10등급. 등급 1에서 시작, 등급 10이 최대치.
- **트럭 EXP = 손님 만족 누적 횟수**. 손님 1명 만족 시 +1 EXP.
- 등급 상승 시 자동으로 신규 유령 1종 + 신규 레시피 1종이 해금됨.
- 도감의 검은 실루엣이 해제되고, 영업 시 신규 유령이 손님 풀에 합류함.

**등급별 해금 매핑:**

| 등급 | 누적 만족 (예시) | 해금 내용 |
|---|---|---|
| 1 (시작) | 0 | 기본 길 잃은 영혼 + 기본 레시피 3종 |
| 2 | 5 | 단골 유령 #1 + 신규 레시피 1 |
| 3 | 12 | 단골 유령 #2 + 신규 레시피 1 |
| 4 | 20 | 단골 유령 #3 + 신규 레시피 1 |
| 5 | 30 | 단골 유령 #4 + 신규 레시피 1 |
| 6 | 42 | 단골 유령 #5 + 신규 레시피 1 |
| 7 | 55 | 단골 유령 #6 + 신규 레시피 1 |
| 8 | 70 | 단골 유령 #7 + 신규 레시피 1 |
| 9 | 95 | 단골 유령 #8 + 신규 레시피 1 |
| 10 (최종) | 120 | 기다리는 할머니 + 최종 레시피 |

> ⚠️ 누적 만족 수치는 1차 설계 예시. 실제 플레이 테스트 후 곡선 조정 예정.

#### 4.4.2 푸드트럭 업그레이드 (Gold)

- 등급 시스템과 별개로, 골드를 소비하여 조리 효율 업그레이드 가능.
- 주요 업그레이드: 슬롯 개수 (2→4), 조리 속도 단축, 손님 수용량 등.
- **등급(Rank)과 업그레이드(Upgrade)는 명확히 분리** — 등급은 콘텐츠 해금, 업그레이드는 효율 향상.

#### 4.4.3 진행 트리거 요약

| 단계 | 트리거 (조건) | 보상 (결과) |
|---|---|---|
| 메인 성장 | 손님 만족 누적 = 트럭 EXP | 트럭 등급 상승 |
| 콘텐츠 해금 | 트럭 등급 상승 | 신규 유령 1종 + 신규 레시피 1종 |
| 서사 보상 | 특정 유물 1개 획득 | 해당 유령 방명록 해금 |
| 기능 보상 | 동일 유물 5개 누적 | 유물 기능적 보상 활성화 |
| 잉여 보상 | 기능 보상 활성화 후 동일 유물 추가 획득 | 골드 +100 자동 변환 |
| 효율 성장 | 골드 소비 | 조리대/슬롯/속도 업그레이드 |

## 5. 도감 및 수집 시스템

### 5.1 유물 시스템 (Artifact)

- 각 유령은 고유 유물 1종을 가짐 (총 10종).
- 유령이 만족하고 떠날 때 확률적으로 유물 드롭.
- **1개째 획득 → 해당 유령의 방명록 해금** (서사 보상).
- **같은 종류 유물 5개 누적 → 기능적 보상 활성화** (영구 패시브).
- **5개 채워진 이후 동일 유물 추가 획득 → 골드 +100으로 자동 변환**.

> ⚠️ 유물 5개 기준은 1차 설계값. 실제 플레이 테스트 후 3~7 범위에서 조정 예정.

### 5.2 10종 유령 손님 및 유물 효과

> ⚠️ 아래 기능적 보상 효과는 1차 설계 예시이며, 실제 조리/영업 시스템 구현 후 밸런스에 맞춰 재정의 예정.

| # | 유령 손님 | 유물 이름 | 기능적 보상 (5개 누적 시) |
|---|---|---|---|
| 1 | 기본 길 잃은 영혼 | 녹슨 나침반 | 유령 손님의 인내심 +5% |
| 2 | 부끄러운 꼬마 유령 | 빛나는 구슬 | 팁 발생 확률 +10% |
| 3 | 바쁜 직장인 영혼 | 낡은 서류가방 | 재료 버튼 클릭 애니메이션 속도 증가 |
| 4 | 미련이 남은 화가 | 빛바랜 이젤 | 레시피 힌트 UI 가독성 향상 |
| 5 | 방랑하는 여행가 | 낡은 지팡이 | 새로운 손님 등장 간격 10% 단축 |
| 6 | 어린 베이커 | 은은한 밀대 | 확률적으로 보너스 효과 (재정의 예정) |
| 7 | 숲을 가꾸는 정령 유령 | 꿈틀거리는 새싹 | 영업 시작 시 보너스 골드 지급 |
| 8 | 밤의 연주자 | 은빛 플루트 | 이탈하려던 손님이 확률적으로 다시 머무름 |
| 9 | 책을 읽는 학자 | 낡은 돋보기 | 유물 드롭 확률 +5% |
| 10 | 기다리는 할머니 | 황금빛 골무 | 모든 요리 판매 가격 +15% |

### 5.3 방명록 시스템

- 해당 유령의 유물을 1개 획득한 시점에 즉시 해금.
- 할머니의 추억과 유령의 사연을 담은 텍스트로 구성.
- 도감 UI에서 유령 카드 탭 시 열람 가능.

### 전용 레시피 (등급 상승으로 해금, 9종)

각 단골 유령의 시그니처. 등급 2~10에서 단골 유령과 함께 해금되며, 11등급에서 최종 레시피(황금빛 약과 타르트)가 해금됩니다.

| # | 레시피 | 재료 (순서대로) | 연관 유령 |
|---|---|---|---|
| 1 | 달콤 크레페 [반죽] - [생크림] - [딸기] - [초코시럽] - [빛나는 구슬] | 부끄러운 꼬마 유령 |
| 2 | 고농축 샷 라떼 | 원두 → 원두 → 우유 → 얼음 → 서류더미 설탕 | 바쁜 직장인 영혼 |
| 3 | 무지개 카나페 | 크래커 → 치즈 → 토마토 → 아보카도 → 삼색 과일 | 미련이 남은 화가 |
| 4 | 영혼 충전 스모어 딥 [크래커] - [초코시럽] - [바나나] - [마시멜로] - [영혼모닥불] | 방랑하는 여행가 |
| 5 | 별 가루 소보로 | 반죽 → 버터 → 설탕 → 아몬드 → 별가루 스프링클 | 어린 베이커 |
| 6 | 숲속 롤케이크 [반죽] - [초코시럽] - [말차크림] - [도토리] - [버터플라이 캔디] | 숲을 가꾸는 정령 유령 |
| 7 | 은빛 선율 와플 | 와플 → 꿀 → 블루베리 → 생크림 → 음표 초콜릿 | 밤의 연주자 |
| 8 | 지혜의 허브 티 | 뜨거운 물 → 찻잎 → 레몬 → 허브 → 돋보기 설탕 | 책을 읽는 학자 |
| 9 | 황금빛 약과 타르트 (최종) | 타르트지 → 조청 → 약과 → 잣 → 금가루 | 기다리는 할머니 |

> 각 레시피의 마지막 재료는 해당 유령의 시그니처 소품과 연결되어 있어, 도감과 방명록의 서사적 연결고리 역할을 합니다.

## 7. 아트 디렉션

별빛 푸드트럭의 비주얼은 **'따뜻한 손글씨 도감'**을 컨셉으로 합니다.

- **스타일**: 수채/손그림 느낌의 2D. 부드러운 외곽선과 은은한 글로우 효과.
- **톤 레퍼런스**: Cozy Grove, Spiritfarer, Stardew Valley 계열의 따뜻한 일러스트 코지 톤.
